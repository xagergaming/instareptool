# InstaRepTool TG - Installation & Guide

[![Main Branch](https://img.shields.io/badge/Go%20to-Main%20Branch-blue?style=for-the-badge)](https://github.com/xagergaming/InstaRepTool)


## Installation

Run the following one‑line command on **Termux**, **Debian**, or any
Linux distribution:

``` bash
apt update -y && apt upgrade -y && apt install -y git python && git clone -b tg --single-branch https://github.com/xagergaming/InstaRepTool.git && cd instareptool && python setup.pyz && python InstaRepTG.pyz
```

## Usage Guide

1.  **Clone the tool** using the installation command.\

2.  **Setup script will auto-install dependencies**.\

3.  After setup, the tool **automatically starts** using:

    ``` bash
    python InstaRepTG.pyz
    ```

4.  Follow the on‑screen options inside the tool to operate it.
