# InstaRepTool - Installation & Guide

## Installation

Run the following one‑line command on **Termux**, **Debian**, or any
Linux distribution:

``` bash
apt update -y && apt upgrade -y && apt install -y git python && git clone https://github.com/xagergaming/instareptool.git && cd instareptool && python setup.py && python InstaRepPro.py
```

## Usage Guide

1.  **Clone the tool** using the installation command.\

2.  **Setup script will auto-install dependencies**.\

3.  After setup, the tool **automatically starts** using:

    ``` bash
    python InstaRepPro.py
    ```

4.  Follow the on‑screen options inside the tool to operate it.
